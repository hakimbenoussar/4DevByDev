Firefox Developer Edition

Pour les développeurs

Ce navigateur a été pensé par des développeurs pour des développeurs et il dispose de nombreux outils de développement. Il est disponible à la fois sur Windows, sur Mac et sur Linux.

Thème sombre

Firefox Developer Edition dispose d’un thème sombre ce qui ressemble à la plupart des éditeurs de texte actuels et qui est moins fatiguant pour les yeux.

Aperçu en mode Responsive

L’aperçu en mode Responsive est une fonctionnalité intéressante, qui vous permet de jouer avec les différentes tailles d'écran sans changer la taille de la fenêtre de votre navigateur (Raccourci Ctrl + Maj + M).

L'éditeur de styles

L'éditeur de styles vous permet d'afficher et de modifier les styles CSS associés à une page. Vous pouvez aussi créer et enregistrer des feuilles de style CSS à partir de pages web existantes.

Modules pour application mobile

Il dispose d’un éditeur de texte WebIDE qui est directement intégré à Firefox Developer Edition (Raccourci Maj + F8).

Une web app est une application mobile développée avec des langages de développement web (HTML5, CSS3, Angular ou React) et exécutée depuis un navigateur web mobile.

Mozilla a également ajouté de nouveaux outils dans Firefox Developer Edition, tels que l'extension Valence, qui vous permet de connecter les outils de développement Firefox à d'autres navigateurs (par exemple Safari sur iOS).

Modules complémentaires

Colorzilla

Colorzilla est un module qui permet de récupérer très rapidement des couleurs sur une page web grâce à l'outil pipette. Il dispose d'une palette de couleur ainsi qu'un historique des couleurs, il peut aussi analyser les sites pour récupérer toutes les couleurs utilisées.

Measure-it

Measure-it permet d'avoir des informations sur la largeur et la hauteur des éléments sur une page web en pixel.

Web developper

Web developper est un outil complet qui permet par exemple de désactiver certains éléments dans une page comme par exemple les styles CSS, les images voir les liens.

Wappalyzer

Wappalyzer permet d'avoir des informations concernant les technologies utilisées sur une page web. Il détecte les systèmes de gestion de contenu (CMS), les Boutiques en ligne, les serveurs web, les Frameworks JavaScript, les Outils de statistiques et beaucoup d'autres choses.

LiveReload

LiveReload permet de synchronisé son éditeur de texte avec la page web et avoir un affichage en temps réel des modifications apporté son fichier de programmation web à chaque sauvegarde.
